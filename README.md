### 簡介

#### 一個關於台灣植物的App，分四個科別，而各科別有四至六種植物。在每種植物裡有詳細的介紹，且標示發現的地點。

運用Xcode實作，Swift語言撰寫。使用StoryBoard的物件，也開啟內建的地圖功能能夠標示出地理位置座標。

此App裡實現了

  * multiple buttons with a single segue：傳遞資訊
  * class：儲存植物資訊
